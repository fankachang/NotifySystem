# Line Notification Service (LNS)

簡短說明：此專案為 Line 訊息通知服務需求規格（LNS）。主要功能包含使用者管理、群組/訂閱管理、訊息類型管理與發送 API，並整合 Line Messaging API 作為發送管道。

快速上手：

1. 初始化（已在此 repo 完成）：

```bash
git clone https://github.com/fankachang/NotifySystem.git
cd NotifySystem
```

2. 建置與執行（範例 - Docker）：

```bash
# 使用 docker-compose 或 podman-compose 啟動
docker-compose up -d --build
```

3. 主要檔案：
- `Prompt/request.md` - 需求規格書（主要內容）
